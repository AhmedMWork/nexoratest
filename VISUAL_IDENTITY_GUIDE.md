# NEXORA V1.1 — Midnight Atelier Visual Identity Guide

## Direction

**Midnight Atelier** is the V1.1 design direction for NEXORA: a premium fashion look built around quiet luxury, dark editorial layouts, warm ivory text, and champagne accents.

## Core Palette

### Dark Mode

- Obsidian Black: `#050505`
- Soft Black: `#0B0B0D`
- Graphite: `#17171A`
- Elevated Graphite: `#202024`
- Warm Ivory: `#F4F0E8`
- Muted Stone: `#B8B0A3`
- Muted Taupe: `#8A8175`
- Champagne Gold: `#C8A96A`

### Light Mode

- Warm Ivory: `#F4F0E8`
- Soft Cream: `#FAF7F0`
- Porcelain: `#FFFDF8`
- Soft Black: `#0B0B0D`
- Stone Border: `#DDD4C7`
- Champagne Gold: `#B9975B`

## Design Rules

1. Use champagne only for calls to action, active states, sale badges, and luxury highlights.
2. Avoid bright orange or generic yellow in the storefront.
3. Product cards should feel editorial: large image, quiet text, minimal badges.
4. Admin UI should stay functional but use the same dark luxury system.
5. No customer-facing Demo, Mock, Test, Fake, or Coming Soon buttons.
6. Admin access is available through `/admin`, the header icon, mobile menu, and footer.

## Main Updated Areas

- Header and mobile menu
- Hero section
- Product cards
- Footer
- Admin login page
- Admin sidebar with View Store shortcut
- Theme system and light mode overrides
- Mobile product grid
