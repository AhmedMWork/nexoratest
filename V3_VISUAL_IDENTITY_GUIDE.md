# NEXORA V3 — Visual Identity Guide

## Direction
Soft Luxury Editorial Commerce.

The V3 visual language is inspired by warm fashion-catalog interfaces: ivory background, sand panels, dust rose accents, taupe borders, and mocha typography.

## Core palette
- Warm Ivory: `#F5F0E8`
- Soft Sand: `#ECE4D8`
- Luxury White: `#FAF7F2`
- Dust Rose Beige: `#CFAE9E`
- Stone Taupe: `#CFC3B7`
- Deep Mocha: `#5C4A42`
- Soft Brown Gray: `#8A7A72`

## Dark mode palette
- Deep Cocoa Black: `#171311`
- Warm Graphite: `#231D1A`
- Muted Beige Text: `#E9DED3`
- Dust Rose Accent: `#C7A191`
- Soft Border Brown: `#4A3D37`

## Principles
- Light-first brand experience.
- Minimal text.
- Editorial spacing.
- Rounded soft panels.
- Gentle animation.
- No technical/debug language visible to customers.
